このページは、AI/solver 実装で採用するルール解釈を一覧できるようにした参照メモです。詳細な変更履歴は rules_decisions.md、進捗は AI_PROGRESS_CHECKLIST.md を参照してください。

重要修正: ミラー対象は「手を降ろす」または「NTP にデバフを付与する」効果だけです。 バフ系、スキップ、必殺スキル、対カウンタースキルは反射できません。

ミラー解釈

ミラー（メイン）は「相手が手を降ろす」または「自分にデバフが付与される」場合だけ、代わりに NTP がその効果を発動します。対象外の効果に対してミラーを宣言した場合、ミラーは消費され、TP の元スキルは通常どおり解決されます。

フラッシュ可条件成立時、NTP が手を2つ降ろす。
セメント可両者の上げた指に基づいて通常どおりセメントを適用する。
ガード不可バフなので反射しない。ミラーは消費され、TP にガードが通常どおり付く。
チャージ不可バフなので反射しない。ミラーは消費され、TP にチャージが通常どおり付く。

ブースト / リバーシ / タイム不可手を降ろす効果でも NTP へのデバフ付与でもないため反射しない。ミラーは消費され、TP の必殺効果は通常どおり解決する。

参照・ストック

項目solver 解釈
コピー直前の数字または参照可能スキルを2回分発動する。
ストック直前の非数字・参照可能スキルだけをストックできる。
ストック集合solver では重複なしの有限集合。すでに持っている同一スキルは追加しない。
チョイスsolver では選択対象を TP action に含める。同時行動モデルを保つため、反応後の隠れた後出し選択にはしない。
オール全ストックを必ず発動する。指定順に不足があれば残りを決定的な順序で補う。
ドロップ自分の現在ストックしているスキル集合を、相手の次フェーズで宣言不可にする。
