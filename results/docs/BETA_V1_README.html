v1ベータ版 遊び方・構成

Complete v1 ベータ版

現行の Complete 版を変更せず、新ルールを独立実装したテスト用分岐です。

起動

Windowsでは Beta_v1/play_beta.bat をダブルクリックします。現行版と同じ形式の対戦画面がブラウザで自動的に開きます。開始前に、ミラーとハンマーをそれぞれON/OFFできます。

ブラウザ版をリポジトリ直下からコマンドで起動する場合：

# http://127.0.0.1:8010 を起動し、ブラウザを自動で開く
python -m Beta_v1.beta_ai.play_server

# ブラウザを自動で開かずサーバーだけ起動
python -m Beta_v1.beta_ai.play_server --no-browser

従来のコマンドプロンプト版も、開発・デバッグ用として個別に残しています：

python -m Beta_v1.play_beta

# ミラーON・ハンマーOFFで直接起動
python -m Beta_v1.play_beta --mirror --no-hammer

# 両方ONで、AIを先手にする
python -m Beta_v1.play_beta --mirror --hammer --human-second

対戦相手AIは、ミラーがONでもミラー（通常）・ミラー（反応）を使用しません。ハンマーは使用します。

実装の分離

  • beta_solver/：ベータ専用の状態、合法手、純粋ルール遷移
  • beta_ai/：ベータ専用の価値ネット、ナッシュ探索、学習・評価
  • models/：ベータ専用学習済みモデルと評価結果
  • data/：ベータ専用学習データ
  • tests/：変更裁定とAI接続の回帰テスト
  • RULES_IMPLEMENTATION.md：実装裁定書

現行版のルール・AI・学習済みモデルは参照も上書きもしていません。対戦画面の見た目と操作感を揃えるため、Complete/complete_ai/webplay の画面資産だけを読み取り専用で共有しています。

AI

実戦AIは、同時宣言をゼロ和行列として解く1手ナッシュ探索と、4つのAB設定を混ぜて学習した価値ネットを組み合わせています。チャージとチョイスは、相手の反応公開後に選べる後出し分岐として探索します。

学習済みモデル：models/value_beta_v1.pt

再学習：

python -m Beta_v1.beta_ai.train_beta --states 4000 --generations 3 --epochs 60

基準AIとの評価：

python -m Beta_v1.beta_ai.evaluate_beta --games-per-seat 10

現在のモデルは、盤面点数を葉に使う未学習1手ナッシュAIに対し、4設定・先後入替の80局で53勝27敗でした。これは短時間ベータ学習の比較結果であり、完全最適性の証明ではありません。
