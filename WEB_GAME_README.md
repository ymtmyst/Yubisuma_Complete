# ブラウザ版ユビスマ

既存の完全ルール版ゲームエンジンを、ブラウザから操作できるプレイ画面です。
AI学習・分析機能とは独立しており、対戦相手の行動は従来どおりランダム戦略です。

## 起動方法（Windows）

`play_web.bat` をダブルクリックしてください。ブラウザで自動的にゲーム画面が開きます。

コマンドから起動する場合:

```powershell
cd Complete
python web_game.py
```

画面が自動で開かない場合は、ブラウザで `http://127.0.0.1:8000` を開いてください。
終了するときは起動した黒い画面で `Ctrl+C` を押します。

## 画面操作

1. 自分のターンでは、宣言する数字またはスキルを選びます。
2. 上げる親指の本数を選ぶと、手の見た目が変わります。
3. 相手ターンでは、相手の宣言が見えないままリアクションと親指を選びます。
4. 「宣言する」または「同時に公開する」でターンを解決します。

スマートフォン幅にも対応しています。同じPC内のブラウザで遊ぶ用途を想定した試作版です。
